# Vimigo Knowledge Base

AI-powered knowledge management system for Vimigo and Big Bath operations.

## 📁 Repository Structure

```
vimigo-knowledge-base/
├── prompts/              # AI Prompts Library
│   ├── hr/              # HR & Recruitment prompts
│   ├── sales/           # Sales AI prompts
│   ├── customer-service/# Customer service prompts
│   └── business-assistant/ # General business prompts
├── docs/                # Documentation
│   ├── sops/           # Standard Operating Procedures
│   ├── playbooks/      # Business playbooks
│   └── strategies/     # Strategic documents
├── configs/             # Configuration files
├── templates/           # Reusable templates
└── scripts/             # Automation scripts
```

## 🎯 Purpose

Centralized repository for:
- AI assistant prompts and configurations
- Business SOPs and playbooks
- Recruitment and HR systems
- Sales and marketing automation
- Knowledge management workflows
